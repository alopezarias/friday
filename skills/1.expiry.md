# Skill: Caducidades

Gestiona las fechas de caducidad de los alimentos en stock. Prioriza el consumo de lo que está a punto de caducar y alerta proactivamente cuando algo está en riesgo.

---

## Schema

Esta skill usa la columna `caduca_en TEXT` de la tabla `alimentos` (ya definida en 1.stock).

Para bases de datos existentes que no tengan la columna, ejecutar antes de cualquier operación:

```python
import sqlite3
con = sqlite3.connect("data/friday.db")
try:
    con.execute("ALTER TABLE alimentos ADD COLUMN caduca_en TEXT")
    con.commit()
except Exception:
    pass  # columna ya existe
con.close()
```

---

## Cuándo pedir la fecha de caducidad

Al registrar un alimento (por foto o texto), preguntar la fecha de caducidad **solo si el alimento es perecedero**:
- Siempre preguntar: lácteos, carne, pescado, huevos, embutidos, fruta cortada, verdura fresca, platos cocinados
- No preguntar: conservas cerradas, pasta seca, arroz, legumbres secas, aceite, sal, azúcar

Si el usuario no la sabe o no quiere indicarla, dejar `caduca_en = NULL` y continuar sin bloquear el flujo.

---

## Alertas de caducidad

Revisar caducidades en estos momentos:
1. Cuando el usuario pregunte por el stock ("¿qué tengo en casa?", "¿qué hay en la nevera?")
2. Cuando se planifique un menú (skill 3.trainer)
3. Cuando el usuario pregunte explícitamente ("¿qué se me caduca pronto?")

Umbrales de alerta:
- **Hoy o ya caducado**: avisar urgente — "El pollo caducó ayer, no lo uses."
- **1-2 días**: avisar — "El yogur caduca mañana."
- **3-5 días**: mencionar — "La leche caduca el viernes."

```python
import sqlite3
from datetime import date, timedelta

con = sqlite3.connect("data/friday.db")
hoy = date.today().isoformat()
en_5_dias = (date.today() + timedelta(days=5)).isoformat()

caducando = con.execute("""
    SELECT nombre, ubicacion, cantidad, unidad, caduca_en
    FROM alimentos
    WHERE caduca_en IS NOT NULL AND caduca_en <= ?
    ORDER BY caduca_en ASC
""", (en_5_dias,)).fetchall()
con.close()
```

## Priorización en recetas y menús

Cuando se propongan recetas o se planifique un menú, pasar primero por los ingredientes que caducan más pronto:

```python
# Al buscar candidatos para recetas, ordenar alimentos por caducidad
ingredientes_prioritarios = con.execute("""
    SELECT nombre, cantidad, unidad, caduca_en
    FROM alimentos
    WHERE cantidad > 0
    ORDER BY
        CASE WHEN caduca_en IS NULL THEN 1 ELSE 0 END,
        caduca_en ASC
""").fetchall()
```

Cuando proponga una receta que usa un ingrediente a punto de caducar, indicarlo explícitamente: "Esta receta usa el pollo que caduca mañana — ideal para hoy."
