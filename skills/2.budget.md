# Skill: Presupuesto de alimentación

Rastrea el gasto en comida. Vinculado a la lista de la compra: cuando se compra algo, se registra el precio. Permite ver cuánto se gasta al mes y en qué.

---

## Schemas

```sql
-- Añadir columna de precio a lista_compra (migración si ya existe)
ALTER TABLE lista_compra ADD COLUMN precio_unitario REAL;
ALTER TABLE lista_compra ADD COLUMN precio_total REAL;

CREATE TABLE IF NOT EXISTS gasto_alimentario (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    fecha TEXT NOT NULL,              -- 'YYYY-MM-DD'
    concepto TEXT NOT NULL,           -- nombre del ítem o 'compra supermercado'
    importe REAL NOT NULL,
    categoria TEXT,                   -- 'proteina' | 'verdura' | 'fruta' | 'lacteo' | 'cereal' | 'otro'
    presupuesto_mensual REAL,         -- solo en registro de configuración (id=1)
    notas TEXT,
    creado_en TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE TABLE IF NOT EXISTS presupuesto_config (
    id INTEGER PRIMARY KEY,           -- siempre id=1
    mensual REAL NOT NULL DEFAULT 300,
    actualizado_en TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

Ejecutar las ALTER TABLE dentro de un try/except (pueden ya existir las columnas).

---

## Configuración del presupuesto

Si no existe `presupuesto_config` o el usuario lo menciona, preguntar el presupuesto mensual para alimentación. Guardarlo con `id=1`.

"¿Cuánto quieres gastar al mes en comida?" — si no sabe, sugerir 250-350€ como rango habitual para una persona.

---

## Registro de gasto

**Flujo principal:** cuando el usuario marca ítems como `comprado` en la lista de la compra, preguntar si quiere registrar el precio:

"¿Cuánto has pagado en total? (o dime los precios por ítem si los tienes)"

Si da precio total → dividir proporcionalmente entre ítems o guardar como entrada única.
Si da precios por ítem → actualizar `precio_unitario` y `precio_total` en `lista_compra` e insertar en `gasto_alimentario`.

**Registro manual:** el usuario también puede decir "he gastado X€ en Y".

```python
import sqlite3
from datetime import date

con = sqlite3.connect("data/friday.db")
con.execute("""
    INSERT INTO gasto_alimentario (fecha, concepto, importe, categoria)
    VALUES (?, ?, ?, ?)
""", (date.today().isoformat(), concepto, importe, categoria))
con.commit()
con.close()
```

## Consultas

**Gasto del mes actual:**
```python
mes = date.today().strftime("%Y-%m")
total = con.execute(
    "SELECT SUM(importe) FROM gasto_alimentario WHERE fecha LIKE ?", (f"{mes}%",)
).fetchone()[0] or 0
```

**Alertas de presupuesto:**
- Al registrar un gasto, comparar acumulado del mes vs. presupuesto
- Si supera el 80%: "Llevas X€ de tus Y€ mensuales — te quedan Z€"
- Si supera el 100%: avisar

**Responder a:**
- "¿Cuánto he gastado este mes?" → total + desglose por categoría
- "¿Cuánto me queda de presupuesto?" → presupuesto - gastado
- "¿En qué gasto más?" → agrupar por categoría
- "¿Cuánto me cuesta esta receta?" → sumar precios de ingredientes desde lista_compra
- "¿Cuál es mi gasto medio semanal?" → total mes ÷ semanas transcurridas
