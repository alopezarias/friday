# Skill: Meal prep (batch cooking)

Analiza el menú semanal planificado y genera un plan de cocinado en bloque: qué preparar de una vez, en qué cantidades, y cómo conservarlo. Objetivo: cocinar una vez, comer bien toda la semana.

---

## Cuándo activar esta skill

- El usuario pide explícitamente: "¿qué preparo el domingo?", "ayúdame a organizar el meal prep", "qué cocino para la semana"
- Tras planificar un menú semanal (3.trainer), si el usuario dice "bien, ¿cuándo lo cocino?"

---

## Lógica de agrupación

Dado un rango de fechas del menú planificado en `menu_diario`:

```python
import sqlite3
from collections import defaultdict

con = sqlite3.connect("data/friday.db")

# Obtener todas las recetas del menú para el período
platos = con.execute("""
    SELECT md.fecha, md.tipo_comida, r.nombre, r.porciones,
           r.tiempo_prep_min, ri.nombre as ing, ri.cantidad, ri.unidad
    FROM menu_diario md
    JOIN recetas r ON md.receta_id = r.id
    JOIN receta_ingredientes ri ON ri.receta_id = r.id
    WHERE md.fecha BETWEEN ? AND ?
    AND md.consumido = 0
    ORDER BY r.nombre
""", (fecha_inicio, fecha_fin)).fetchall()

# Agrupar: cuántas veces aparece cada receta → multiplicar ingredientes
recetas_agrupadas = defaultdict(int)
for plato in platos:
    recetas_agrupadas[plato['nombre']] += 1  # N veces en el menú

```

## Reglas de conservación

Usar este conocimiento para indicar cómo conservar cada preparación:

| tipo de plato | nevera (días) | congelador (meses) |
|--------------|--------------|-------------------|
| arroz / pasta cocidos | 3-4 días | 2 meses |
| carne cocinada | 3-4 días | 3 meses |
| pescado cocinado | 2 días | 2 meses |
| legumbres cocidas | 4-5 días | 3 meses |
| verdura salteada | 3-4 días | 2 meses |
| sopas / cremas | 4-5 días | 3 meses |
| platos con huevo | 2 días | no congelar |
| ensaladas | no preparar con antelación | — |

**Regla clave:** si un plato aparece más de 3-4 días en el menú, dividir en lote nevera (primeros días) y lote congelador (días posteriores).

## Presentación del plan de meal prep

```
MEAL PREP — semana [fecha inicio] al [fecha fin]

Cocinar el [día sugerido, ej: domingo]:

1. [Nombre receta] × [N raciones]
   Ingredientes totales: Xg pollo, Xml aceite...
   Tiempo: ~X min
   Conservar: X días en nevera / X meses en congelador

2. [Nombre receta] × [N raciones]
   ...

Orden sugerido (lo que más tarda, primero):
  1. Legumbres (poner a remojo la noche anterior)
  2. Arroz + pollo (~45 min)
  3. Verduras (~15 min)

Platos a hacer en el momento (no preparar con antelación):
  - [ensaladas, huevos, etc.]

Tiempo total estimado: ~X horas
```

## Vinculación con lista de la compra

Tras generar el plan de meal prep, verificar si todos los ingredientes necesarios están en stock (`alimentos`) o en la lista de la compra (`lista_compra` con estado `pendiente` o `en_carrito`).

Si faltan ingredientes no contemplados aún, añadirlos a `lista_compra` con `motivo='mealprep: [nombre receta]'`.
